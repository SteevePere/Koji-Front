# Koji-Front

Koji-Front displays Blog Posts from https://jsonplaceholder.typicode.com.

## Run it (development mode)

### 1. Run `npm install`

Installs dependencies.

### 2. Run `npm start`

Runs the app in the development mode.<br />

### 3. Your default browser should open the App Homepage automatically. If not, go to [http://localhost:3000/posts](http://localhost:3000/posts).
